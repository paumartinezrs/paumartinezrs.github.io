Implementation of a Data Structure

This project consisted of the implementation of a KD-Tree. These are data structures used to organize points in a k-dimensional space, particularly useful for facilitating efficient operations like search. To achieve this, it recursively divides the space along the axes into smaller regions, creating a hierarchical binary tree structure.

KD-trees, in low to moderate dimensions, are particularly efficient for nearest-neighbor searches. During a search, the tree is traversed to find the point closest to a given query point. This is done by recursively exploring the branches of the tree and prioritizing those that are more likely to contain the nearest neighbor.

Technologies involved
  • C++

Use of Artificial Intelligence Algorithms

The goal of this project was, through the application of artificial intelligence algorithms, specifically Hill Climbing and Simulated Annealing, to develop the optimal bike transport planning for the Bicing service between different urban stations.

The objective is to collect bikes from stations with low demand forecasts and transport them to those where they are needed most urgently, all while minimizing the costs associated with transport between stations.

Technologies involved
  • Java
  • Local search algorithms

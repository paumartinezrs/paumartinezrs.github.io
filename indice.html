Implementación de una estructura de datos

Este proyecto consitió en la implementación de un KD-Tree. Estas son estructuras de datos utilizadas para organizar puntos en un espacio k-dimensional realmente útiles para facilitar operaciones como la búsqueda de manera eficiente. Para conseguirlo divide recursivamente el espacio a lo largo de los ejes en regiones más pequeñas, creando una estructura de árbol binario jerárquico.

Los árboles KD, en dimensiones bajas a moderadas, son particularmente eficientes para búsquedas de vecinos más cercanos. Durante una búsqueda, el árbol es recorrido para encontrar el punto más cercano a un punto de consulta dado. Esto se hace explorando recursivamente las ramas del árbol y priorizando aquellas que tienen más probabilidades de contener el vecino más cercano.

Tecnologías implicadas
  • C++

Uso de algoritmos de inteligencia artificial

El objetivo de este proyecto fue, mediante la aplicación de algoritmos de inteligencia artificial, específicamente Hill Climbing y Simulated Annealing, desarrollar la planificación óptima del transporte de bicicletas del servicio Bicing entre las diferentes estaciones urbanas.

El objetivo es recolectar bicicletas de estaciones con baja previsión de demanda y trasladarlas a aquellas donde se necesiten con mayor urgencia, todo ello mientras se busca minimizar los costes asociados al transporte entre estaciones.

Tecnologías implicadas
  • Java
  • Algoritmos de búsqueda local
